# Spinning Wheel Promotion Feature

This project is a simple implementation of a **Spinning Wheel** promotional feature for an e-commerce website. Users can spin the wheel to win a discount coupon and apply it to their cart.

## Live Demo
https://spinning-wheel-test.vercel.app

## Tech Stack
- React
- TypeScript
- Vite

## Installation & Run

### Prerequisites
- Node.js (v16 or later)
- npm / yarn / pnpm

### Steps
1. Clone the repository.

2. Install dependencies:
cd spinning-wheel
npm install

3. Run the development server:
npm run dev

4. Open the browser and visit:
http://localhost:5173

## Build
To build the project for production:
npm run build
